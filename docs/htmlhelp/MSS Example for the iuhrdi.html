MSS Example for the Nintendo Wii

Discussion

The MSS SDK supplies MilesWii.CPP, an example program for the Wii. To build this example with CodeWarrior, just create a new project and include the MilesWii.CPP source file. Then add any necessary MSS libraries MSSWii.A, FLTWii.A, MP3Wii.A, OGGWii.A, MSSSRS.A, and MSSDOLBY.A to the project. (Only the MSSWii.A library is mandatory.)

We also ship the Miles for Wii example pre-built, so that you can try it immediately. To use it, simply launch the example image on the Wii with ndrun. Make sure to your DvdRoot emulation drive has the media directory in it.
