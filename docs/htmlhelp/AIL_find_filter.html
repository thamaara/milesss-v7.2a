Miles Sound System SDK 7.2a

AIL_find_filter

The function AIL_find_filter tries to locate a filter by its name.

S32 AIL_find_filter(
  char * name,
  HPROVIDER *found
);

In Parameters

name
contains the name of the filter to search for.

Out Parameters

found
contains the found provider if the filter name was located.

Returns

0 if no filter was found; non-zero if a filter was found.

Discussion

You can call this function before AIL_open_filter or AIL_set_sample_processor to locate a specific filter to use. You can enumerate all of the filters with the AIL_enumerate_filters function.

The HPROVIDER handle values returned by this function can change from one program execution to the next. For example, the high-pass filter may have the HPROVIDER handle value "1" during one execution, and the value "2" the next. You must, therefore, always use these functions to obtain the proper HPROVIDER handle value at runtime.
