Miles Sound System SDK 7.2a

Resource Usage on the Playstation 3

Discussion

Miles is very low impact on the PS3. Currently, we do all work on the PPU on a background thread. We use about 10% of the PPU to play 32 samples right now.

However, in an upcoming version, we will move most of that down to the SPU. We expect mixing, ADPCM decompression, MP3 decompression and DSP filtering to run on the SPU.

For our sound output, we use one of the four output ports that libAudio supports so we should run along other middleware just fine.
