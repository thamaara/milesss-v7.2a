Memory management under PlayStation 2

Discussion

By default on the PS2, Miles simply calls the malloc/free C runtime library functions.

You can, of course, intercept these memory allocation requests with the AIL_mem_use_malloc and AIL_mem_use_free functions.

Miles can also use IOP and SPU RAM for storing audio data. To do this, first you load the sound data into IOP or SPU memory (the RAD_IOPQueueAsyncReadAtTop function does this), and then you use the Miles streaming audio functions (such as AIL_open_stream) to play it.
