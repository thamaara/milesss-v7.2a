MSS Example for the PlayStation 3

Discussion

The MSS SDK supplies MilesPS3.CPP, an example program for the PS3. To build this example with SN Systems or CodeWarrior, just create a new project and include the appropriate .CPP source file (demo.cpp). Then add any necessary MSS libraries MSSPS3.A, FLTPS3.A, MP3PS3.A, and OGGPS3.A to the project. (Only the MSSPS3.A library is mandatory; you can edit the input format types to remove the other libraries).

We also ship the Miles for PS3 example pre-built, so that you can try it immediately. To use it, simply launch the example image on the PS3 with the files in the media directory copied into the HOST_ROOT/media/miles directory.
