Miles Sound System SDK 7.2a

MSS Example for the PlayStation 2

Discussion

The MSS SDK supplies MilesPS2.CPP, an example program for the PS2. To build this example with SN Systems or CodeWarrior, just create a new project and include the appropriate MilesPS2.CPP source file. Then add any necessary MSS libraries MSSPS2.A, FLTPS2.A, MP3PS2.A, OGGPS2.A, MSSDTS.A, and MSSDOLBY.A to the project. (Only the MSSPS2.A library is mandatory; you can edit the #defines near the top of MilesPS2.CPP to take advantage of the optional Miles features supported by the DTS and Dolby Surround libraries.)

We also ship the Miles for PS2 example pre-built, so that you can try it immediately. To use it, simply launch the example image on the PS2 with two parameters on the command line. The first command line parameter should be a host0 specific path to the directory containing the IOP runtime modules (the *.IRX files) - this is usually "c:/usr/local/sce/iop/modules". The second command line parameter should be the path to the directory containing the Miles PS2 example media files - usually "host0:c:/MilesPS2/examples/media".

Note that the first parameter shouldn't use "host0:" on the front, because it always loads the IRX files from host0. The second parameter does need a device first (like host0:, or cd0:, etc.), since it can play them off any device for testing purposes.
