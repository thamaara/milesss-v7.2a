Miles Sound System SDK 7.2a

MILES_DIGITAL_FLAGS

The basic type MILES_DIGITAL_FLAGS describes the layout of digital audio.

Fundamental Values

 Value  Description
 DIG_PCM_ORDER If present, stereo data is in RL order; else LR.


Group: Digital Audio Services
Related Functions: AIL_init_sample
