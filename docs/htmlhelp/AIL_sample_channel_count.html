Miles Sound System SDK 7.2a

AIL_sample_channel_count

The function AIL_sample_channel_count returns information about the source channels associated with a sample.

S32 AIL_sample_channel_count(
  HSAMPLE S,
  U32 *dwChannelMask
);

In Parameters

S
indicates the sample whose channel information is to be returned.

Out Parameters

dwChannelMask
if not NULL, receives the Microsoft WDM-compatible channel mask for the sample.

Returns

The number of source channels associated with the HSAMPLE.

Discussion

This function returns 1 for mono samples, 2 for stereo samples, or the actual number of channels in a multichannel .WAV or Ogg Vorbis sample.

For a detailed discussion of the dwChannelMask parameter, see the How do I use multi-channel sound output in Miles? FAQ
