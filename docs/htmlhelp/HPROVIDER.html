HPROVIDER

The basic type HPROVIDER is a handle to an available RIB provider.

Discussion

HPROVIDER handles are returned from the AIL_enumerate_filters, AIL_find_filter, AIL_set_sample_processor, AIL_set_digital_driver_processor, AIL_sample_processor, AIL_digital_driver_processor, and AIL_digital_output_filter functions.
