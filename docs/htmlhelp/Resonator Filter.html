Resonator Filter

Discussion

This filter provides a resonant filter that allows frequencies around the center to pass. The advantage of this filter is that you can "slide" the center and achieve some interesting effects.

Name for use with AIL_find_filter: "Resonator EQ Filter"

Properties:

"Resonator Center": Center Frequency. Range: 20.0 Hz to Nyquist (one-half the driver's output sample rate in Hz).

"Resonator Width": Width of Frequency. Range: 20.0 Hz to Nyquist (one-half the driver's output sample rate in Hz). Hz.

"Mix": Wet/dry mix. Range: 0.0 (no effect) to 1.0 (full effect).

When a mix value of 0.0 is specified, the filter will operate on an internal "fast path" designed to consume minimal CPU resources.
