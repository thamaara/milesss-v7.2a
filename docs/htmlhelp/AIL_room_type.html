Miles Sound System SDK 7.2a

AIL_room_type

The function AIL_room_type returns the current room type.

S32 AIL_room_type(
  HDIGDRIVER driver
);

In Parameters

driver
indicates the driver to query.

Returns

One of the MILES_ROOM_TYPE constants.

Discussion

If the provider doesn't support room types, this function will return -1.
