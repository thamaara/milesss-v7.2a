Miles Example for Xbox 360

Discussion

The Miles SDK supplies a simple example program for Xbox that demonstrates most of the handy Miles features. To build this example in an IDE-based environment, just create a new project, include MilesXenon.CPP to the project, and then add the Miles libraries (MSSXenon.lib, OggXenon.lib, and MP3Xenon.lib to the project).

The Xbox example file is found in the \src\examples directory. You should also copy the "\examples\media" directory to "xe:\media" on the Xbox.

We also ship the Miles for Xbox 360 example pre-built, so that you can try it immediately. To use it, copy the MilesXenon.exe file into the root directory of the 360, and then make a folder on the Xbox called "media" and copy the contents of the Miles media directory inside.
