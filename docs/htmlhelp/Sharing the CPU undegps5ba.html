Sharing the CPU under Xbox 360

Discussion

On the Xbox 360, the work that Miles does (MP3 decompression, for example) is done on a background thread on another CPU. You don't have to do anything special to share the CPU - everything is handled automatically. If you need to associate a particular CPU core with the background thread used by Miles, see the AIL_CPU_AFFINITY preference for more information.
