Miles SRS Circle Surround Provider

Discussion

The SRS Circle Surround 6.1 Encoder for MILES Sound System ("SRS Miles Encoder") is a run-time surround sound encoder. Using home-theater and gaming surround sound systems, this provider can play audio in breathtaking 6.1 surround sound, allowing real (non-simulated) 3D sound positioning. The SRS Miles Encoder is also based on the Miles API which makes it extremely fast! The Miles SRS Circle Surround provider is accessed by passing MSS_MC_SRS_CIRCLE_SURROUND as the channel specification in a call to AIL_open_digital_driver or AIL_quick_startup.

Note that using the SRS Miles Encoder does not give you the right to use the SRS Circle Surround Logo. To use this, you must become an SRS licensee (the criteria to do this are listed below).

Overview of SRS Circle Surround

SRS Circle Surround encoding provides the capability to encode up to 6.1 channels of audio for transmission or storage over two output channels or standard two-channel carriers. This represents a substantial improvement over previous Lt/Rt based encoding systems, which are not as well suited to handle modern, multichannel media with more than four channels of audio. Circle Surround encoding is backward compatible with all matrix decoders, such as Dolby Pro Logic and Dolby Pro Logic II, with playback performance subject to the limitations of the specific decoder.

For linear tracks, a Circle Surround Encoder, model CSE-07, and Circle Surround Decoder, model number CSD-07, are used. There are also plug-ins available for those users of VST and TDM based sytems. These include the SRS Circle Surround TDM Pro, model number TDM-02, and the SRS Circle Surround VST Pro, model number VST-02.

On Becoming an SRS Labs, Inc. Licensee

In order to become an SRS Circle Surround Licensee, which grants you the right to use the SRS Circle Surround Logo, four basic criteria need to be met:

I. You must use an SRS-approved, real-time encoding scheme, such as the Miles SRS Circle Surround 6.1 Encoder.

II. You must request the necessary trademarks from SRS Labs, Inc.

III. You must sign a royalty-free License and Trademark Usage Agreement, which grants a license to use the SRS Circle Surround Trademark, subject to these conditions.

IV. You must send a sample of the finished title prior to the time of release.
