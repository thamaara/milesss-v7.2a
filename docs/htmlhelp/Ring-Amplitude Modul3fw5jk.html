Ring-Amplitude Modulator Filter

Discussion

This filter uses a sine wave to modulate (change) the signal. You can create "robot" type voices or underwater type effects with lots of motion with this filter.

Name for use with AIL_find_filter: "Ring Modulator Filter"

Properties:

"Ring Mod Freq": Frequency of modulation. Range: 20.0 Hz to Nyquist (one-half the driver's output sample rate in Hz).

"Ring Mod Phase Left": Phase shift in radians applied to left channel. Range: 0.0 to 2*pi

"Ring Mod Phase Right": Phase shift in radians applied to right channel. Range: 0.0 to 2*pi

"Mix": Mix of modulated signal. Range: 0.0 (original - dry) to 1.0 (full modulation - wet).

When a mix value of 0.0 is specified, the filter will operate on an internal "fast path" designed to consume minimal CPU resources.
