Miles Sound System SDK 7.2a

AIL_sample_stage_property

The function AIL_sample_stage_property retrieves and/or sets the current value of the specified sample-level property.

S32 AIL_sample_stage_property(
  HSAMPLE S,
  SAMPLESTAGE pipeline_stage,
  char const * name,
  void *before_value,
  void const * new_value,
  void *after_value
);

In Parameters

S
specifies the sample whose filter should be addressed.
pipeline_stage
specifies the stage to address. Use one of the SAMPLESTAGE constants.
name
specifies the name of the property to retrieve.
new_value
is a pointer to a new value to set. If you don't want to set the property, pass in 0.

Out Parameters

before_value
is a pointer to a variable that will contain the property value before changing it. Always use this pointer to retrieve read-only properties. You can use 0 if you don't want to retrieve the prior value.
after_value
is a pointer to a variable that will contain the property value after changing it. You can use 0 if you don't want to retrieve the post-value.

Returns

Non-zero for success, 0 for failure.

Discussion

Sample-level properties are properties that are read from a single sample. For properties that are global to the filter, use the AIL_filter_property function.

This function can retrieve or set the current value of a property.

Example files that use this function: filter.cpp, multiflt.cpp
