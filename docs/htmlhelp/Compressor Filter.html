Compressor Filter

Discussion

This filter "compresses" the dynamic range of a signal by making quiet signals louder and loud signals a bit quieter.

Name for use with AIL_find_filter: "Compressor Filter"

Properties:

"Compressor Ratio": Amount of compression. Range: 1.0 to 10.0 compression ratio.

"Compressor Gain": Legacy parameter, concatenated with "Mix" (below). Range: 0.0 to 1.0

"Mix": Mix of the compressed signal. Range: 0.0 (original - dry) to 1.0 (full compress - wet).

When a mix value of 0.0 is specified, the filter will operate on an internal "fast path" designed to consume minimal CPU resources.
